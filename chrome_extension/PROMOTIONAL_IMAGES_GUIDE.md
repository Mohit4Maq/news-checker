# Promotional Images & Screenshots Guide

This guide will help you create all the promotional images and screenshots needed for Chrome Web Store submission.

## Required Images

### 1. Small Promotional Tile (REQUIRED)
- **Size:** 440 x 280 pixels
- **Format:** PNG or JPG
- **Purpose:** Displayed in Chrome Web Store search results
- **What to include:**
  - Extension icon (large, centered or top-left)
  - Extension name: "News Checker"
  - Tagline: "AI-Powered Fact-Checking"
  - Key benefit: "One-Click News Analysis"
  - Clean, professional design

### 2. Screenshots (1-5 images)
- **Size:** 1280 x 800 pixels OR 640 x 400 pixels
- **Format:** PNG or JPG
- **Purpose:** Show extension in action
- **Recommended screenshots:**
  1. Extension popup interface
  2. Analysis results in Streamlit app
  3. Extension icon in browser toolbar
  4. Content extraction working
  5. Before/after comparison

### 3. Large Promotional Tile (Optional but Recommended)
- **Size:** 1400 x 560 pixels
- **Format:** PNG or JPG
- **Purpose:** Featured placement in Chrome Web Store

### 4. Marquee Promotional Tile (Optional)
- **Size:** 920 x 680 pixels
- **Format:** PNG or JPG
- **Purpose:** Special promotional placement

## Design Guidelines

### Colors
- **Primary:** Use colors from your extension icon
- **Background:** Light, clean background (white or light gray)
- **Text:** Dark, readable (black or dark gray)
- **Accents:** Use your brand colors sparingly

### Typography
- **Title:** Bold, large (24-32px)
- **Subtitle:** Medium (16-20px)
- **Body:** Regular (12-14px)
- **Font:** Use clean, modern fonts (Arial, Helvetica, Roboto, Open Sans)

### Layout
- Keep it simple and uncluttered
- Use white space effectively
- Center important elements
- Make text readable at small sizes

### Branding
- Include extension icon prominently
- Use consistent colors
- Maintain professional appearance
- Show value proposition clearly

## What to Include in Each Image

### Promotional Tile Should Show:
1. Extension icon (large, clear)
2. Extension name
3. Main benefit/feature
4. Visual appeal (not too cluttered)

### Screenshots Should Show:
1. **Screenshot 1:** Extension popup with URL and "Analyze" button
2. **Screenshot 2:** Streamlit app showing analysis results
3. **Screenshot 3:** Extension working on a news website
4. **Screenshot 4:** Key features (fact-checking, bias detection)
5. **Screenshot 5:** Before/after or comparison view

## Step-by-Step: Creating Promotional Tile

1. **Open design tool** (Canva, Figma, or HTML template)
2. **Set canvas size:** 440 x 280 pixels
3. **Add background:** Light color or gradient
4. **Add extension icon:** 128x128 or larger, centered or top-left
5. **Add text:**
   - Title: "News Checker" (bold, 28px)
   - Subtitle: "AI-Powered Fact-Checking" (18px)
   - Tagline: "One-Click News Analysis" (14px)
6. **Add decorative elements:** Subtle borders, shadows, or icons
7. **Export as PNG** (high quality)

## Step-by-Step: Creating Screenshots

### Screenshot 1: Extension Popup
1. Open extension popup in Chrome
2. Use browser dev tools to set viewport to 1280x800
3. Take screenshot
4. Optionally add annotations or highlights

### Screenshot 2: Analysis Results
1. Open Streamlit app with analysis results
2. Set browser viewport to 1280x800
3. Take screenshot
4. Highlight key features with arrows or boxes

### Screenshot 3: Extension in Action
1. Open a news article
2. Show extension icon in toolbar
3. Show popup open
4. Take screenshot

## Tips for Best Results

1. **Keep it simple** - Don't overcrowd images
2. **Use high contrast** - Text should be easily readable
3. **Show value** - Highlight main benefits
4. **Be consistent** - Use same colors/fonts across all images
5. **Test at small sizes** - Promotional tiles appear small in store
6. **Use real screenshots** - Show actual extension, not mockups
7. **Add annotations** - Arrows and labels help explain features
8. **Optimize file size** - Compress images for faster loading

## File Naming Convention

- `promotional-tile-440x280.png`
- `screenshot-1-popup.png`
- `screenshot-2-analysis.png`
- `screenshot-3-action.png`
- `large-promotional-1400x560.png`

- Clear, readable text
